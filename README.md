Установка проекта : 
npm install/
yarn install

Запуск проекта :
npm run serve/
yarn serve

Сборка продакшн-версии : 
npm run build-prod

Стек технологий : 
 Язык разработки - javascript ES6+/typescript v3.6.3. Изначально проект разрабатывался с помощью jinja2, но после того как заказчик потребовал динамические фильтры на страницах поиска было решено переписать его на react v.16.0.0 + styled(https://www.styled-components.com/) . Предпочтения отдавались react hooks и компонентному стилю разработки, компоненты-классы почти отсутствуют. В качестве сборщика использовался parcel v.1.12.3.   
Основные папки и компоненты :
src/assets - статика
src/router - routes.tsx содержит все основные пути.
src/routes - лежат компоненты для routes.tsx
src/api - лежат  все обертки и пути для получения данных с помощью fetch
src/styles - стили
src/components/graph  - граф на странице пользователя ( /user/:id ). Строится с помощью библиотеки cytoscape. На данный момент данные с бека поменялись, но внутри компонента ожидаются старые данные. При больших данных загрузка фризит страницу, поэтому строится при помощи web-workerа в папке src/web-workers.
/store - хранилище данных + роутинг при помощи mobx и mobx-react-router.
